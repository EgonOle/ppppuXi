ppppuXi Cutie Mark Crusaders Character Pack v0.1 (Aug 31, 2016)
The contents of the zip contains the Cutie Mark Crusaders from My Little Pony: Friendship is Magic as mods for ppppuXi. The original flash files for these characters were created by GeneralScot.
